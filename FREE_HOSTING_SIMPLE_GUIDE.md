# 🆓 FREE HOSTING - Super Simple Guide

## ✅ SAB READY HAI! Tum bas 3 clicks karo:

---

## 🎯 STEP 1: GitHub Account (2 minutes)

```
1. Browser me jao: https://github.com/signup

2. Form bharo:
   - Email: tumhara email
   - Password: strong password (8+ characters)
   - Username: kuch bhi unique (example: foodapp2024)

3. Email verify karo

✅ Account ban gaya!
```

---

## 🎯 STEP 2: Backend Deploy - Railway (5 minutes)

### A. Railway Account:
```
1. Jao: https://railway.app

2. "Login with GitHub" button dabao

3. GitHub se authorize karo

✅ Railway account ban gaya!
```

### B. Deploy Backend:
```
1. "New Project" button dabao

2. "Deploy from GitHub repo" select karo

3. "Configure GitHub App" dabao

4. Repository access do:
   - "Only select repositories" choose karo
   - Tumhara repo select karo
   - "Save" dabao

5. Railway dashboard me wapas aao

6. "Deploy from GitHub repo" fir se try karo

7. Apna repo select karo

8. "Deploy Now" dabao

✅ Backend deploy ho raha hai! (5 min wait karo)
```

### C. Domain Generate Karo:
```
1. Railway dashboard me "Settings" tab

2. Scroll down → "Networking" section

3. "Generate Domain" button dabao

4. URL copy karo:
   Example: food-app-production.up.railway.app

✅ Backend live!
```

---

## 🎯 STEP 3: Frontend Deploy - Vercel (5 minutes)

### A. Vercel Account:
```
1. Jao: https://vercel.com

2. "Sign Up" → "Continue with GitHub"

3. Authorize karo

✅ Vercel account ban gaya!
```

### B. Deploy Frontend:
```
1. "Add New" → "Project" dabao

2. GitHub repo import karo (same repo)

3. Configure:
   - Framework: "Other" select karo
   - Root Directory: "frontend" type karo
   - Build Command: npx expo export -p web
   - Output Directory: dist

4. Environment Variable add karo:
   Name: EXPO_PUBLIC_BACKEND_URL
   Value: https://YOUR-RAILWAY-URL.up.railway.app
   (Railway se copy kiya hua URL)

5. "Deploy" button dabao

✅ Frontend deploy ho raha hai! (10 min wait karo)
```
